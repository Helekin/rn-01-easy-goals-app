## Getting Started

### Prerequisites

- Use any code editor

### Installation

1. Clone the repo

```sh
git clone https://github.com/Helekin/nodejs-01-easy-expressjs.git
```

2. Install dependencies

```sh
expo install
```

3. Run the application

```sh
npx expo start
```
